## 0. 时间尺度（所有公式共用）

| 量 | 值 | 说明 |
|----|-----|------|
| `REAL_TICK` | 0.35 s | 主循环间隔（1× 速度） |
| `GAME_SECONDS_PER_REAL_SECOND` | 60 | 1 真实秒 ≈ 1 游戏分钟 |
| `GAME_DAY_SECONDS` | 86400 gs | 1 游戏日 |
| **1 游戏日（1×）** | **≈ 24 真实分钟** | 1440 真实秒 |

食草动物吃 **1 块活草** 的耗时：

- 羊 / 鹿：`SHEEP_EAT_GAME_SECONDS = 5400` gs → **90 游戏分钟** → 每头 **16 块草/日**
- 兔（现状）：`RABBIT_EAT_TICK_SECONDS = 5` 真实秒 → 每头 **≈288 块/日**（**严重失衡，必须改**）

## 1. 设计目标（封闭生态）

**不是**「永远静止」，而是 **10~20 游戏日** 内可观察的波动：

| 指标 | 目标 |
|------|------|
| 活草数量 | 常态 8~18，不长期为 0 |
| 羊 / 鹿 | 可繁殖，不全灭，总量有上限 |
| 兔 | 作为小型猎物底盘，**不**单独吃光全图草 |
| 狼 | 有肉时不连杀；绝食 **个体** 死亡，不整包迁走 |
| 尸体 / 生肉 / 熟肉 | 无指数堆积 |
| 玩家 | 可打猎，但 `PLAYER_CAMP_MEAT_RESERVE` 抑制过度狩猎 |

**玩家肉需求**（影响生态，须纳入预算）：

- `PLAYER_HUNGER_PER_GAME_DAY = 100`：不进食约 1 日饿满
- `PLAYER_CAMP_MEAT_RESERVE = 4`：家域熟肉 ≥4 时不再主动打猎（饿到 need 除外）
- 主链前期玩家会打猎 → 设计时预留 **0.5~1 头羊当量/日** 的玩家捕食压力（波动大，trace 观测）

**预期**：消耗 80~100、恢复能力 120~180（受河岸格与上限限制），常态 **12~16 块活草**。

## 3. 动物初始值与上限（步骤 1）

### 3.1 初始种群

| 卡 | 数量 | 性别 | 初始状态 | 位置原则 |
|----|------|------|----------|----------|
| `sheep` | 2 | 1 公 1 母 | 找草 | 营地西北林缘，远离玩家出生点 |
| `rabbit` | 2 | — | 找食 | 营地南侧草地 |
| `wolf` | 2 | 1 公 1 母 | 游荡，hunger≈15 | 营地北侧，**非** 0 饥饿 |
| `deer` | 2 | 1 公 1 母 | 找草 | 营地西侧河谷，**必须写 sex** |
| `taoyuanElder` 等 | 各 1 | — | 远观/边界 | `taoyuan_edge` 动态找格 spawn |

**禁用 live 入口**（代码保留，封闭 trace 不触发）：

- `_update_sheep_attraction`（外羊迁入）
- `_migrate_wolf_pack_away` / `_return_wolf_pair`（狼群整包迁出/召回）
- `_check_wolf_sex_replacement`（入口补性别）
- 羊无草 → **不再** 走向 `MAP_EXIT` 删除

### 3.2 种群上限（硬上限 + 繁殖门控）

| 物种 | 成体上限 | 幼体上限 | 繁殖条件摘要 |
|------|---------|---------|-------------|
| 羊 | 4 | 1 `lamb` | 草 ≥3、无狼 4 格内、已有公母 |
| 鹿 | 4 | 1 `deerFawn` | 草 ≥4、无狼 5 格内、公母、总数 <5 |
| 兔 | 6 | 0（步骤 1 不繁殖） | 步骤 2 启用 |
| 狼 | 4 成 | 1 `wolfCub` | 狼窝存在、窝容量、公母、有肉喂幼狼 |

羊羔成长：`age` 每 12 tick 周期 +1，≥60 转成年（**保持现状** ≈ 5 游戏日）。

### 3.3 狼：捕食与绝食

| 参数 | 现值 | **建议** |
|------|------|---------|
| `WOLF_MEAT_PER_DAY` | 2 | **2**（保持） |
| `WOLF_FAMINE_LEAVE_DAYS` | 3 | **停用整包迁出** |
| `WOLF_ADULT_STARVE_GRACE_DAYS` | 3 | **3**：`daysWithoutMeat` 超限 → **该狼个体** remove |
| `WOLF_DIGEST_SECONDS` | 28 | **28**（保持） |
| 猎杀优先级 | `largePrey -2` 分 | **步骤 1 修正**：成鹿 **不** 享受 largePrey 优先；幼鹿/羊羔/兔/田鼠（步骤 2）用 `juvenile` / 距离 |

**捕猎评分（步骤 1 最小改法）**：

```text
score = distance
- 若目标是 deer 且 can_reproduce(deer): 不减分（当成难猎物）
- 若目标是 lamb / deerFawn / rabbit: score -= 1.5
- 若目标是 stunned 或幼体: 保持现有 -1
```

即：**狼优先易猎物，不无脑追成年鹿**。

### 3.4 无草时食草动物行为

| 物种 | 行为 |
|------|------|
| 羊 | `state=等草`，向最近 `grassland`/`riverbank` 格移动，**不** 出图 |
| 鹿 | 同羊，且 `惧火`/`躲人` 优先 |
| 兔 | 保持 `游荡`，步骤 2 改向灌木 |

## 5. 验收 trace（步骤 1）

**closed_ecology_trace**（新建或补强）跑 **≥2000 tick**（约 11 游戏日 @1×），每 200 tick 采样：

- `grass`, `sheep`, `lamb`, `rabbit`, `deer`, `deerFawn`, `wolf`, `wolfCub`
- `corpse`, `raw_meat`, `cookmeat`
- `hunt_events`, `repro_events`, `starve_deaths`

**PASS**：

- 草从未连续 400 tick 为 0
- 羊+鹿不同时为 0（除非玩家杀光）
- 狼 `meat_peak` 与尸体数无爆炸
- 至少 1 次 `deerFawn` 产生或 trace 注明「条件未满足但门控正确」
- 无 `MAP_EXIT` 羊离开事件

**L2b**：1×，`USELESS=0`，蘑菇棚建成。
