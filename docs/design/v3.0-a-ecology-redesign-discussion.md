# v3.0-A 生态重思考：小型植物丛 + 兔鼠底盘 + 狐狸生态位

> 状态：讨论稿，给用户与 Cursor 对齐方向  
> 类型：生态机制设计，不是立即执行单  
> 背景：当前 `deer` 接入后体感像“肉更多的羊”，没有形成生态本质增量。后续新增卡必须改变世界循环，而不是增加换皮分类。

---

## 1. 核心判断

增量阶段的新增卡必须是机制增量。

一张新卡进入游戏后，应该改变至少一个循环：

- 资源消耗 / 恢复循环
- 繁殖 / 捕食循环
- 栖息地 / 领域循环
- 玩家干预反馈
- 社会观察反馈

如果新增卡只是“另一种肉”“另一种猎物”“另一种更高数值目标”，则不算有效增量。

当前鹿的问题不只是“没有幼鹿”，而是生态位不清：它没有显著改变草地、狼、玩家、桃源人之间的判断。

---

## 2. 封闭生态前提

桃源不是普通开放野外。本轮应按封闭生态理解：

- 羊、兔、狼、鹿、狐狸、田鼠、草、灌木等一开始就在同一生态环境中。
- 不应主要靠“地图外迁入 / 离开地图”维持数量平衡。
- 数量平衡应来自初始数量、资源恢复、繁殖速度、捕食压力、藏身地、窝、风险和上限。

调试目标不是静止平衡，而是合理波动：

- 草地资源不长期清零。
- 小型猎物不会爆炸。
- 捕食者不会无限猎杀。
- 尸体 / 肉不会爆炸。
- 玩家干预会改变局部生态结构。

---

## 3. 小型植物丛是生态底盘

讨论后认为，生态不应只从动物开始，而应先建立“小型植物丛”。

### 3.1 灌木丛 `bush`

`bush / 灌木丛` 不是装饰卡，也不是单纯浆果来源。它应是一个复合生态节点：

```text
灌木丛
= 植物资源
+ 浆果产能
+ 微型生物宿主
+ 小型动物掩体
+ 狐狸窝候选
+ 可再生生态节点
```

建议标签：

```text
plant.patch
cover.small
berry.source
microfauna.host
den.candidate.fox
regenerates
```

建议能力：

```text
capability.produce_resource
capability.regenerate
capability.provide_cover
capability.host_microfauna
capability.support_den
```

第一版作用：

- 周期性产出或恢复浆果。
- 内部维护蚯蚓 / 甲虫数量。
- 给田鼠 / 兔子 / 狐狸提供隐藏和加速。
- 可被狐狸家庭占据或转化为狐窝前身。

### 3.2 蚯蚓 / 甲虫

第一版不建议做独立牌，而作为草丛 / 灌木丛的内部状态：

```text
worm_count
beetle_count
microfauna_regen_timer
```

理由：

- 当前它们主要承担底层食物数量，不是玩家直接操作对象。
- 作为植物丛内部状态，能更快形成生态循环。
- 未来若要做钓鱼、堆肥、鸡鸭、药材等，可以再把蚯蚓 / 甲虫升级成可见卡。

这不违背“万物皆卡”，因为当前承载它们的是“小型植物丛”这张卡。

---

## 4. 小型猎物底盘：兔子 + 田鼠

### 4.1 兔子定位

兔子是快速繁殖的小型基础猎物。

特点：

- 繁殖快。
- 消耗草 / 浆果。
- 是狐狸、狼等捕食者的低风险猎物。
- 数量过多会对草地 / 灌木资源造成压力。

兔子不应只是“低级动物”，而应是生态底盘之一。

### 4.2 田鼠 `fieldMouse`

田鼠是兔子之外的第二种小型基础猎物。

它和兔子的区别：

- 田鼠在草丛和灌木丛之间穿梭。
- 繁殖能力强，接近兔子。
- 消耗很小，主要吃蚯蚓 / 甲虫，也可少量吃浆果。
- 强依赖小型植物丛覆盖。
- 是狐狸主食之一。

建议标签：

```text
being
animal
smallPrey
herbivore 或 omnivore.small
burrower
cover_user
```

建议能力：

```text
capability.move
capability.forage_microfauna
capability.reproduce
capability.flee
capability.use_cover
capability.be_hunted
```

田鼠的机制价值：

- 支撑狐狸，不让狐狸只靠兔子。
- 减少兔子被单一捕食者吃灭的风险。
- 让草丛 / 灌木丛的内部资源有意义。

---

## 5. 狐狸生态位

狐狸不是小号狼。

### 5.1 基本定位

狐狸 = 小家庭捕食者 + 小型植物丛潜行者 + 清道夫。

它的主要生态作用：

- 压制兔子和田鼠数量。
- 偷吃尸体 / 搬运肉，减少尸体堆积。
- 避让狼、人、火域。
- 依赖灌木丛 / 草丛进行隐藏、加速和筑窝。

### 5.2 狐狸家庭 vs 狼群

狼：

- 无论多少，倾向建到同一狼窝。
- 是群体 / pack 逻辑。

狐狸：

- 以小家庭为单位。
- 一公一母 + 幼狐。
- 每个家庭占据一个灌木丛或狐窝。
- 多个狐窝可以存在，但彼此保持距离。
- 不合并成大群。

这能让“窝”本身产生生态差异。

### 5.3 狐狸行为优先级

第一版建议：

1. 捕田鼠。
2. 捕兔子。
3. 若附近有尸体且狼 / 玩家不在旁边，偷偷搬运一部分肉回窝。
4. 兔鼠不足且狐狸饥饿时，少量威胁幼鹿 / 羊羔。
5. 避让狼、人、火域，不挑战成年羊 / 成年鹿 / 玩家。

### 5.4 小型植物丛给狐狸的特点

狐狸在草丛 / 灌木丛附近：

- 移动更快，或行动冷却更短。
- 更难被狼 / 玩家发现。
- 更容易伏击兔子 / 田鼠。
- 可以把肉藏回狐窝。

这才是狐狸的机制特点，而不是数值换皮。

## 6. 建议新增卡

若以此方向作为下一次生态修正切片，建议新增：

| 卡 | 作用 |
|---|---|
| `bush` 灌木丛 | 小型植物丛、浆果、微型生物、掩体、狐窝前身 |
| `fieldMouse` 田鼠 | 低消耗高繁殖小型猎物，依赖草丛/灌木 |
| `fox` 狐狸 | 兔鼠专精捕食者，清道夫，小家庭单位 |
| `foxCub` 幼狐 | 狐狸繁殖结果，跟随家庭成长 |
| `foxDen` 狐窝 | 可由灌木丛占据/转化，狐狸家庭中心 |

鹿修正仍需要做：

- `deerFawn` 幼鹿。
- 鹿繁殖。
- 鹿高速逃跑、偏草地 / 河岸 / 林缘。
- 成年鹿不应被普通狼无脑优先捕猎。

但生态底盘更应优先关注：

```text
小型植物丛
→ 兔子 / 田鼠
→ 狐狸
→ 狼
→ 玩家干预
```

---

## 7. 捕猎评分原则

当前“狼更优先 largePrey 鹿”的逻辑不符合生态。

捕猎者应按以下朴素模型选择目标：

```text
收益 × 成功率 - 风险 - 追逐成本
```

建议拆成标签 / 权重：

- `prey.small`
- `prey.medium`
- `prey.large`
- `prey.juvenile_vulnerable`
- `escape.fast_runner`
- `defense.herd_resist`
- `meat.low`
- `meat.medium`
- `meat.high`
- `predation.rabbit_specialist`
- `predation.rodent_specialist`
- `diet.carnivore`
- `diet.omnivore`
- `diet.herbivore`

第一版不一定全部落地，但方向应是能力 / 风险 / 成功率，而不是简单按肉量排序。

## 8. 可能的下一执行切片

建议不要直接进入 v3.0-B。先把 v3.0-A 生态修正为真正的机制增量。

候选切片名：

```text
v3.0-A2.2 小型植物丛与狐狸生态位
```

范围建议：

1. 新增 `bush`，具备浆果产能、微型生物内部状态、再生能力、cover 标签。
2. 新增 `fieldMouse`，依赖草丛 / 灌木，吃微型生物，快速繁殖。
3. 新增 `fox` / `foxCub` / `foxDen`，狐狸以小家庭为单位，主食田鼠和兔子。
4. 狐狸能偷尸体肉回窝，形成清道夫行为。
5. 狼压制狐狸，狐狸避让狼 / 人 / 火域。
6. 修正捕猎评分，不再让狼因 `largePrey` 无脑优先成年鹿。
7. 增加 closed ecology trace，观察草、灌木、兔、田鼠、狐狸、狼、鹿、尸体/肉在封闭地图中的波动。

不做：

- 完整季节系统。
- 完整土壤肥力系统。
- 鱼、鸟、野猪等更多动物线。
- 旅人解封。
- 桃源人交易 / 雇佣。
