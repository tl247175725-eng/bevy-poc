# 生态多样性 + 溢出乐趣：设计定稿

> 2026-06-06

---

## 不做什么

- 不模拟细节（几只虫、多少天孵化）
- 不加复杂系统（季节、天气、湿度）
- 不加独立卡满天飞

## 做什么

- 每张新卡消除一个生态单点故障
- 每个新标签连接一条行为因果
- 每个宿主用容纳承载子卡，不散落满地
- 同一产出多种取法——溢出乐趣

---

## 一、封闭闭环验证

封闭生态只需要三层：生产者+消费者+分解者。我们在此基础上加了五层——每一层都对应一个真实的生态位。

```
能量入口
  ├── 草皮（河岸再生）—— 被羊/鹿/兔/水牛吃掉
  ├── 树林（栎树、松树）—— 产橡子/松塔（不依赖草）
  ├── 水藻（水潭/河流）—— 被虫/贝/鱼吃掉
  └── 灌木（浆果+栖虫）—— 被兔/鼠吃掉

消费层
  ├── 大型食草：羊（群居·flocking）、鹿（机警）、水牛（成体不可猎·tough）
  ├── 小型食草：兔（高产·prolific）、田鼠（掘穴·burrower）、竹鼠（林地掘根）
  ├── 地栖鸟类：雉鸡（散居·flocking，吃虫吃种子）
  ├── 鱼类：鱼（水生·aquatic，仅水格可活动）
  ├── 贝类：贝（固着·sessile，浅水分布）
  └── 人类：狩猎采集者（靠鱼+猎物+采集 60/30/10 比例）

捕食层
  ├── 狼（群猎·pack_hunter，单狼不猎大型猎物）
  └── 狐（多策略·mesopredator，限杀2只/天，清腐）

分解层
  ├── 尸体自然腐解 → humus → 草加速恢复
  ├── 狐清腐取肉
  ├── 虫群被尸体吸引 → 加速分解 → 为鼠/鸟提供食物
  └── 生肉有腐坏时限（perishable）

迁入物种（不受山谷地理限制）
  ├── 虫（飞行迁入，被尸体吸引）
  ├── 鱼（经暗河水系游入）
  ├── 鸟（飞行迁入，吃虫吃种子，排泄传种）
  └── 种子（风/水/鸟粪携带）
```

## 二、溢出乐趣：一个源头，多种取法

### 树（树林）

```
树林
  ├─ 容纳：【栎树】【松树】
  ├─ 自然掉落：橡子、松塔偶尔落到邻格
  ├─ 采集：点树林→摘橡子/松塔
  ├─ 砍伐：斧头→出木头×N
  └─ 栖息：【鸟巢】
```

### 水（水潭+河流）

```
水潭
  ├─ 容纳：【鱼】【贝】【菱角】【莲子】
  ├─ 捕鱼：矛刺→鱼肉（主动）
  ├─ 鱼筌：陷阱→被动捕鱼
  ├─ 鱼繁殖：水面偶尔跃鱼（视觉溢出）
  ├─ 采菱角：直接摘/自然脱落漂岸边
  ├─ 挖藕：水底、冬季
  └─ 摸贝：浅水、徒手

河流
  └─ 钓/刺：同水潭机制
```

### 地（林下+岩壁）

```
林下格
  ├─ 容纳：【野山药】（地下，全年可挖）
  ├─ 山药藤蔓延→邻格新生（自然溢出）
  └─ 竹鼠拱出山药→人白捡（动物溢出）

岩壁格
  ├─ 锤敲→碎石（主动）
  └─ 自然风化→偶尔掉碎石（被动溢出）
```

### 尸体

```
尸体
  ├─ 自然腐解→humus→草恢复加速
  ├─ 狐清腐→取肉（清道夫）
  ├─ 虫群吸引→加速分解+喂鼠鸟
  └─ 人屠宰→全肉+无humus（人为切断分解链）
```

### 灌木

```
灌木
  ├─ 采浆果（主动）
  ├─ 藏田鼠（生态容纳）
  └─ 狐狸占为窝（转化容纳）
```

## 三、未接的标签行为（待Cursor地基完成后处理）

| 标签 | 所在的卡 | 要接的行为 |
|------|---------|----------|
| `aquatic` | fish | 仅水格移动，离水不动但存活，水桶可养 |
| `sessile` | shellfish | 不移动，仅浅水 |
| `burrower` | fieldMouse, bambooRat | 威胁→underground→不可猎 |
| `nut_producer` | oak | 季节触发→橡子入容纳 |
| `floating` | waterCaltrop, lotus | 水格→水面附着，不占陆地 |
| `tuber` | wildYam | 地下格→可挖，地面生态不影响 |
| `wildPrey` | deer | 更大逃跑半径 |
| `perishable` | 所有meat | 时限到→自毁 |

## 四、与现有系统的复用关系

| 溢出机制 | 复用的现有系统 |
|---------|-------------|
| 橡子/松塔掉落 | natural_drop（同树枝掉落） |
| 树容纳栎树/松树 | containment（同狼窝容纳狼） |
| 水潭容纳鱼/贝/菱角 | containment（同水潭逻辑） |
| 林下容纳野山药 | underground_state（同狼窝地下状态） |
| 斧砍树出木头 | impact 冲击系统（同石头敲碎石） |
| 尸体虫群加速分解 | humus 腐解倒计时 |
| 生肉腐坏 | perishable tick（同尸体腐解倒计时） |
