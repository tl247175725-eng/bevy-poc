## 总原则

每一层都要满足：

1. **可玩闭环** — 玩家能玩到一个完整小循环  
2. **只加 1～2 个「旧桃源没有的东西」**  
3. **验收用感受** — 不堆功能清单  
4. **可停手** — 本层完成即可发布/试玩，不依赖下层  

**四条不做：** 不做完整 50 人社会再开玩 · 不做完整意向引擎再开玩 · 不做 4～6 倍地图一次铺完 · 不做复合弓/盐业/娱乐/枢纽同期上线  

**三条并行线（每版只主推一条）：** 世界事实线 · 社会线 · 商人线
