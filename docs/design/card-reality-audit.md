# 卡片真实世界抽象审查报告

> 方法论 §3.1：用抽象逻辑模拟真实复杂系统。不是用细节去模拟细节——是用最本质的因果标签去抽象。
> 每张卡只提炼一个核心动作：**真实世界的它干了什么，连带了什么，被什么干了。**

---

## 一、非动物卡的因果链

### 草皮 — 本质：被吃→再生
草的真实因果不是"生长速度多快"，而是**它连接了太阳和所有食草动物**。这是我们生态的能量入口。
- **破缺**：草归零累计 tick 已出现但不显示严重度。没有任何卡因为草归零而触发人口调节——真实里草荒→食草动物迁移或死亡。

### 灌木 — 本质：栖虫→藏鼠→被狐狸占
灌木的真实因果不是"有几个浆果"，而是**它是田鼠的生存屏障和狐狸的建筑材料**。
- **当前标签够了吗**：`cover.small + den.candidate.fox` 表达了核心关系。`berry.source` 和 `microfauna.host` 是孤立标签——缺的是**行为管线用标签查询来驱动再生**，而不是用常量定时器。

### 尸体 — 本质：死→腐→归土
真实因果是循环的闭合点。动物死→肉被吃或烂掉→肥土→草长。当前尸体和草之间的腐殖土链已通，但**动物意识到"这里有腐肉"靠的是狐的清腐能力，不是所有清腐者都能感知尸体。** 缺一个 `carcass_detector` 类标签。

### 生肉 — 本质：离体→腐坏
真实因果：肉离开身体后，不处理就会烂。当前 7-12 块生肉堆地上无限久。
- **缺标签**：`perishable`——有此标签的物品，N tick 后自毁或触发腐解状态。

### 火 — 本质：热→安全+烹饪+干草
真实因果：火是营地的"引力中心"。动物惧火不是怕受伤——是数百万年进化出的本能回避。
- **正确用法**：`camp.anchor` 标签已经表达了"引力中心"，动物惧火判定的 `FIRE_FEAR_RANGE` 正确。但干草的产生过度依赖火——真实里旱季太阳暴晒也能产干草。留到季节系统。

### 石头/树枝/木头 — 本质：可采集→可加工→可消耗
这些是资源链的起点。真实因果是**它们在环境中"可被感知"**——人看到石头知道能做刀。
- **正确用法**：这些资源卡本身不带"工具"属性——属性是由加工关系赋予的。标签系统通过 `material.stone + material.lumber` 表达了这个抽象层。正确。

## 二、动物卡的群习性

### 羊 — 本质：群居→安全→繁殖
羊的群性不是"几只羊一起走"，而是**孤羊不繁殖**。这是真实因果：零散个体处于持续应激状态。
- **缺标签**：`flocking`——同类少于 N 只 → 行为改变（紧张、不繁殖、不敢吃草）。

### 鹿 — 本质：独居/小群→机警→择地
鹿和羊不同——鹿不群居，但母鹿带崽。公鹿独处。鹿的机警来自 `wildPrey`（但标签孤立）。
- **缺标签**：`vigilant`——有威胁的更早预警半径。不是群性，是警觉性。

### 狼 — 本质：配对群居→协作→领地
狼不单独猎大型猎物。真实里一匹狼猎鹿的成功率极低，两匹才有效。
- **缺标签**：`pack_hunter`——同类 < 2 → 只猎兔鼠级别猎物，不尝试鹿/羊。
- **当前行为**：我们的狼已经是两头一起行动和繁殖，但**没有标签显式表达这种约束**。

### 兔 — 本质：独处/散居→高繁殖→天敌驱动
兔子不群居，但高密度。真实的兔子不靠群体防御——靠数量淹没捕食者。
- **缺标签**：`prolific`——繁殖频率高、每胎多崽。这是兔子所有行为的根本原因。天敌多不是"悲剧"——兔子的生态策略就是被吃。

### 田鼠 — 本质：独处→掘穴→灌丛依赖
田鼠的真实因果：没有灌丛就没有田鼠。掘穴是第二道防线。
- **缺标签**：`burrower` 已注册但行为未激活。鼠的灌丛依赖已通过 `cover_user + use_cover` 表达。

### 狐狸 — 本质：独居配对→多策略→惧狼
狐的群性是"一公一母一窝崽"，不是群猎。它的生存策略是**不被饿死的方法多**——猎鼠、猎兔、清腐。
- **标签表达充分**：`mesopredator + scavenger + cover_user` 已覆盖多策略。惧狼关系正确。

### 水牛 — 本质：小群→大型→亲水→成体不可猎
水牛的群性表现为集体移动，但成体的防御力来自体型而非群性。
- **标签表达充分**：`tough + body.large` 已阻止成体被猎。群性可后续补。

## 三、生态边界

### 封闭原则
这是一个封闭的山谷生态。除了不受地理限制的物种，大部分卡不迁入——生态靠内部循环自持。

**可迁入（不受地理限制）**：
- 鸟 —— 飞行，跨域
- 虫 —— 飞行或风携
- 鱼 —— 经暗河水系游入
- 种子 —— 风/水/动物携带

**不可迁入（受地理限制）**：
所有陆地动物（羊鹿狼狐兔鼠水牛）、所有植物、所有资源。

### 地下水系修改
暗河源头从"河沟"改为**暗河源头水潭**。水潭是静水面，与河岸流动水不同——是小型水生生态的支点。

## 四、生态位缺失

### 分解者链不完整
当前：尸体→腐殖土→草加速。只有两极，缺中间层。
- **缺**：虫类（尸体→虫子→田鼠可食）。真实里腐烂尸体吸引昆虫，昆虫养活了鼠和鸟。这个缺失导致了田鼠食物来源单薄。

### 河流生态完全空白
我们有河、有水桶、有暗河。河流里没有任何生物。
- **缺**：鱼？蛙？河对两岸动物的饮水行为？当前河流只是一个"地形标签"，不是生态位。

### 种子传播机制缺失
草和灌木怎么扩张？当前是定时刷新。
- **缺**：动物吃草→种子通过粪便传播→新的草在远处出现。这个因果关系完整但没实现。

### 没有攀禽/猛禽
树上没有任何鸟。鸟既是猎物也是种子传播者。
- **缺**：鸟对生态来说是"农药"——吃虫、传种、被狐猎。不需要复杂的鸟，一个标签化的小型掠食者即可。

## 四、生态位缺失

基于封闭边界，以下生态位是可迁入物种填充的空缺：

**水生层（暗河水系）**：
- 水潭→暗河→出口。可迁入：鱼、水生虫。
- 鱼吃水生虫→被鸟/水牛（饮水时）接触。
- 当前河流是纯地形标签，无任何生物。这是一个完整的空白层。

**空中层**：
- 鸟可迁入。吃虫、吃种子、树栖、被狐偶然猎。
- 鸟粪→种子传播→草/灌木在远处生长。

**分解者中间层**：
- 虫（飞行迁入）。尸体吸引虫子，虫子养活田鼠和鸟。
- 当前 humus 直接跳过了虫这一环。

**种子传播**：
- 当前草和灌木靠定时刷新。真实里是风/动物/鸟类传种。
- 可迁入种子 + 鸟类携种 = 植物自然扩散。不需要系统定时刷草。

## 六、溢出乐趣：一个源头，多种取法

**原则：一张卡不只有一个交互方式。同一个产出可以通过不同的方式获得——主动采集、被动掉落、工具加工、自然转化。**
**玩家体验到的不是"资源点"，而是"这里有东西在发生"。**

### 已有的溢出

| 宿主 | 取法一 | 取法二 | 取法三 |
|------|--------|--------|--------|
| 树林 | 斧头砍→木头 | 自然掉落→树枝在地上 | 人捡树枝 |
| 灌木 | 采浆果 | 田鼠藏在里面 | 狐狸占为窝 |
| 尸体 | 自然腐解→humus | 狐清腐→肉 | 人屠宰→全肉+无humus |
| 草皮 | 动物吃 | 火边变干草 | 鸟粪传种→新草远处出现 |

### 溢出清单规则

溢出不是刷资源。溢出有三个前提：
1. **有源头宿主**——不是凭空出现，一定来自某张卡
2. **有合理因果**——橡子不是魔法刷新，是秋季成熟+重力掉落
3. **复用已有系统**——掉落用 natural_drop 机制，容纳用 containment 系统，采集用现有交互

| 物种 | 群性 | 本质因果 | 当前状态 |
|------|------|---------|---------|
| 羊 | 群居 | 孤羊不繁殖 | 缺 `flocking` |
| 鹿 | 独居小群 | 母带崽、公独处 | 缺 `vigilant` |
| 狼 | 配对群猎 | 单狼不猎大猎物 | 缺 `pack_hunter` |
| 兔 | 散居高密度 | 靠数量淹没天敌 | 缺 `prolific` |
| 田鼠 | 独居灌丛 | 灌丛=生存 | 缺 `burrower` 行为 |
| 狐 | 配对家庭 | 一公一母一窝 | 标签够 |
| 水牛 | 小群 | 成体不可猎 | 标签够 |

## 五、审查结论

**不是"加细节"。是补齐因果链中缺失的本质标签，让现有系统自洽。**

| 优先级 | 领域 | 缺失 | 后果 |
|--------|------|------|------|
| P0 | 羊 | `flocking` 标签 | 孤羊不繁殖——这是真实因果，不是细节 |
| P0 | 狼 | `pack_hunter` 标签 | 单狼猎鹿成功率极低——驱动力来自群性 |
| P0 | 兔 | `prolific` 标签 | 兔靠数量生存——不繁殖=灭绝 |
| P1 | 生肉 | `perishable` 标签 | 肉堆地上无限久——缺腐解因果 |
| P1 | 田鼠 | `burrower` 行为激活 | 鼠的第二防线——掘穴减少被猎 |
| P1 | 尸体 | 腐解时间差异化 | 小尸体 vs 大尸体——当前统一 15s |
| P2 | 鹿 | `vigilant` 标签 | 更高警觉探索半径 |
| P2 | 分解者 | 缺昆虫卡 | 尸体→虫→鼠的中间层 |
| P2 | 河流 | 缺水生卡 | 河流是地形不是生态 |
