## 1. 定位修正：从"远观NPC"到"流动狩猎采集部落"

原 v3.0-A 的桃源三性格将桃源人定义为"社会远观原型"——静态站在 taoyuan_edge、对玩家保持距离。这个定位不完整。

### 1.1 他们是谁

桃源人是一个封闭河谷内的**狩猎采集部落**，规模约 8~12 人（最终 50），具有以下核心特征：

- **流动**：不在固定聚落定居。在河谷内根据资源消耗/恢复周期轮换活动区域。
- **狩猎采集为主**：环境不适合农业革命——没有大麦小麦，可选作物极少。
- **补充种植**：野生山药、小块根茎类辅助，不是主食来源，不形成农业社会。
- **深度生态知识**：比玩家更懂这个封闭世界——知道哪种灌木快结果、哪个河段有鱼、草地何时恢复。

### 1.2 为什么是狩猎采集

不是设计偷懒不做农业，而是**世界规则排除了农业**：

- 河谷没有可驯化的高热量谷物。
- 河流与山地限制了可耕作面积。
- 封闭生态的承载量天然倾向于低密度人口。
- 狩猎采集是此环境下生存效率最高的方式。

这解释了"为什么只有 8~12 人"——不是还没做 50 人，是生态承载力决定的自然上限。

### 1.3 流动性

桃源人不静止站在 taoyuan_edge。他们的活动逻辑：

```
当前活动区资源衰减
→ 集体向恢复中的区域迁移
→ 不同地形对应不同资源策略：
  - 河岸：取水、捕鱼（未来）、采集水边植物
  - 林缘：采集浆果、山药、小规模狩猎
  - 草地：狩猎羊/兔/鹿
  - 山脚：石器原料、特殊药材（未来）
```

流动不是随机走——是**对环境信号的可见响应**。玩家看他们在哪，就能推断哪片区域的资源在恢复。

## 2. 陌生人法则："铁律"的因果链

> 世界铁律：桃源人不会让任何一个人走出桃花源。外来者一旦误入，即困于小世界；「离开」不是早期选项。
> —— `意向系统与世界演化_v0.3.md` §8

### 2.1 铁律的根源

铁律不是恶意——是封闭部落的生存本能：

- 离开者会暴露桃源的位置，引来外部威胁。
- 部落规模小，每个人都承载着生存知识，不能流失。
- 这条规则在封闭期内是绝对的——不分善恶、不分对象。

### 2.2 陌生人命运：因素挤压，而非好感度

陌生人进入后，命运不由单一"好感度"决定，而是以下因素挤压出的结果：

| 因素层 | 内容 |
|--------|------|
| **资源压力** | 当前食物/水/庇护是否紧缺？资源配置越紧，接纳越难 |
| **身份判断** | 武器携带者 → 威胁 · 伤弱 → 负担 · 带新工具/知识 → 潜在价值 |
| **角色倾向** | 长者偏排斥（守边界）· 采集者看重对方是否懂生态 · 青年对差异好奇 |
| **随机性** | 附近是否有狼？当天是否刚猎到食物？谁先发现陌生人？ |

同一条件两次运行，结果可能不同——这才是故事生成器。

### 2.3 可能的结果（非穷举）

- **死于自然**：被狼杀、饥饿、伤病——桃源人不主动加害，但也不救助。
- **死于资源竞争**：桃源人判断此人消耗太大，放任其死亡。
- **死于部落裁决**：对方携带武器且具威胁，部落主动击杀。
- **边缘寄居**：可在某区域活动，但不得进入部落聚落范围。
- **条件接纳**：能提供有价值劳动（懂编织、识药材等），被部分接纳。
- **融入**：极罕见，需要长期。玩家卡的路线也属此类。

### 2.4 玩家不是普通陌生人

玩家卡的特殊性在于**有能力提供新东西**（商品、工具、服务、组织方式），并且**能自己生存**。这意味着：

- 铁律同样适用于玩家——他在初期不能离开。
- 但玩家不需要被"接纳"——他自己建立营地。
- 玩家的路径是：从边缘观察者 → 被好奇的对象 → 因有用而被容忍 → 因提供价值而改变部落观念。
- 中期商业体系建立后，**玩家是铁律松动的唯一推动力**。

### 2.5 铁律的松绑

铁律不是永久规则。中期通商后：

```
玩家商业体系建立
→ 部分桃源人尝到新商品/新服务的甜头
→ 观念从"封死边界" → "外界可能有更多好东西"
→ 铁律软化：通商路径可被接受，但物理出境仍非轻易
→ 旅人/商队角色从"囚禁体"升级为"商机载体"
```

这个转变本身就是一个重大游戏事件。

## 3. 内部社会结构（最小版）

### 3.1 三种角色的社会含义

| 角色 | 社会位置 | 核心倾向 |
|------|---------|---------|
| **长者** | 传统守护者 | 最坚定的边界维护者、对改变最警惕、拥有生态知识权威 |
| **采集者** | 资源管理者 | 实际判断哪片区域、哪种资源可取用、对浪费敏感 |
| **青年** | 边界试探者 | 对差异好奇、最先接受新事物、但服从长者权威 |

三者之间不是好感度关系，而是**群体决策张力**：长者的拒斥 vs 青年的好奇 vs 采集者的务实判断。

### 3.2 未来的社会扩展

当人口扩展至 8~12 人阶段时补充：
- 猎户、手艺人、妇女/主妇角色
- 亲族关系
- 分工雏形

## 4. 生态知识与环境互动

桃源人是玩家理解世界的"信号灯"：

- 采集者出现在河岸 → 水边资源在恢复
- 长者在林缘停留 → 可能是禁区（危险或神圣）
- 青年靠近玩家营地 → 对外来者的好奇心在上升
- 三人突然离开某片区域 → 资源枯竭或有危险

这不需要玩家"查看状态"——只需要观察他们的位置变化。

## 5. 实现建议

### 5.1 归属切片

建议归入 **v3.0-B** 或 **v3.3 小社会**，不在步骤 1/2 的生态修正中合入。

### 5.2 依赖

- 步骤 1 完成（桃源人 live 可见）
- 步骤 2 完成（生态底盘足够支撑社会行为）
- 社会行为代码框架：流动性逻辑、资源评估判断、角色差异行为

### 5.3 不做

- 不在此切片做 50 人社会系统
- 不做复杂意向引擎
- 不做完整陌生人事件系统（铁律与命运机制属于 v3.3+）
- 不做通商后观念变化（那是腊肉 + 中期系统的事）
