## 生产链

```
wood/twig + waterbucket → wetWood (+ bucket)
wetWood --30s 湿润--> mushroomWood
mushroomWood --impact×2--> mushroom（wood 完全消耗）
hut + mushroomWood → mushroomGreenhouse
```

## 蘑菇农

- 最多 1 人；有棚 2 游戏天后边缘生成
- 招募：玩家相邻 → 面板；拒 3 天内不再来
- 管理需求：wood×2 近棚 / 熟肉 2 天 / 棚边第二 hut+火 / 狼≤4 格 / 饥饿
